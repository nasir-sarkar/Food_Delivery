# Title
Khuda Lagche - Food Delivery Management System

# Quick Look
Khuda Lagche is a basic food ordering system developed in Java. It allows two types of users: Admin and User. Users must register before logging in, while admins can log in directly. The system enables users to place food orders and complete online payments, while admins can manage user accounts. The application features a graphical user interface (GUI) built with Java Swing, providing a simple and interactive experience for users.

The system provides a step-by-step process for ordering food. Users first select a meal type, such as Breakfast, Lunch, or Dinner, and then choose food items from the available menu. Once selections are made, users proceed to complete their orders with an online payment system. Admins, on the other hand, have privileges to manage user accounts, including adding new users, viewing all registered users, and deleting accounts when necessary. This project is designed as a foundational-level application for understanding Java GUI development and basic user management.
